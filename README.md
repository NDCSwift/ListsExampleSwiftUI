# 📝 SwiftUI Lists Example

A focused SwiftUI demo on the `List` view — covering static and dynamic lists, custom row views, swipe actions, and `ForEach` with identifiable data.

---

## 🤔 What this is

This project dives specifically into SwiftUI's `List` component: rendering arrays of data with `ForEach`, building custom row views, adding swipe-to-delete and swipe actions, grouping with `Section`, and using `Identifiable` conformance for stable list identity.

## ✅ Why you'd use it

- **Dynamic List + ForEach** — rendering arrays with `Identifiable` items
- **Custom row views** — extracting row content into reusable sub-views
- **Swipe actions** — `.swipeActions` modifier with destructive and non-destructive actions
- **Sections** — grouping list rows with `Section` headers and footers
- **onDelete / onMove** — enabling edit mode with drag-to-reorder and delete

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/ListsExampleSwiftUI.git
cd ListsExampleSwiftUI
```

### 2. Open in Xcode
- Double-click `ListsExampleSwiftUI.xcodeproj`

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team**

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique identifier (e.g., `com.yourname.ListsExample`).

---

## 🛠️ Notes

- If you see a code signing error, check that Team and Bundle ID are set.

## 📦 Requirements

- iOS 16+
- Xcode 15+
- Swift 5.9+
